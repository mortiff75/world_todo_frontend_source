## Installation

Install my-project with yarn

```bash
  npm i .
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/mortiff75/real_state.git
```

Go to the project directory

```bash
  npm start
```

### that will be start with wbpack-dev-sever

Install dependencies

```bash
  yarn add
```

Start the api/app.js

```bash
for each side:
  npm run dev
```
